# Fonctionnalités

## Pour les entreprises clientes (TPE/PME)

- **Assistant d'onboarding** : Inscription multi-étapes
- **Recherche de prestataires** : Matching géographique & sectoriel
- **Demandes d'adhésion** : Postuler auprès des prestataires médicaux
- **Gestion des employés** : Ajouter, modifier, supprimer
- **Système de réservation** : Planifier des rendez-vous médicaux
- **Accès aux documents** : Consulter les dossiers médicaux

## Pour les entreprises médicales

- **Tableau de bord** : Vue d'ensemble des clients & demandes
- **Gestion des demandes** : Accepter/refuser les candidatures
- **Liste des entreprises clientes** : Toutes les TPE/PME affiliées
- **Dossiers employés** : Accès aux données de santé
- **Gestion des rendez-vous** : Planifier & suivre les consultations

## Pour les employés

- **Portail personnel** : Consulter son dossier médical
- **Visibilité des rendez-vous** : Voir les consultations planifiées
- **Accès aux documents** : Télécharger les documents médicaux

## Fonctionnalités système

- **Accès basé sur les rôles** : Personnel médical, admin entreprise, employé
- **Multi-tenant** : Support de multiples entreprises médicales & clientes
- **Système d'invitation** : Invitations par email pour nouveaux utilisateurs
- **Workflows de statut** : En attente → Accepté/Refusé pour les demandes
