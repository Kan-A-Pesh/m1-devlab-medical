# Stack Technique

## Frontend

| Technologie | Usage |
|-------------|-------|
| **Next.js 16** | Framework React avec App Router |
| **React 19** | Librairie UI |
| **TypeScript** | Typage statique |
| **Tailwind CSS v4** | Styles |
| **shadcn/ui** | Composants (primitives Radix) |
| **Motion** | Animations |

## Backend

| Technologie | Usage |
|-------------|-------|
| **oRPC** | RPC typé (remplace REST/GraphQL) |
| **Better Auth** | Authentification |
| **Zod** | Validation de schémas |

## Base de données

| Technologie | Usage |
|-------------|-------|
| **PostgreSQL** | Base de données principale |
| **Drizzle ORM** | Accès typé à la base |

## Infrastructure

| Technologie | Usage |
|-------------|-------|
| **Vercel** | Hébergement & déploiement |
| **Docker** | PostgreSQL local |

## Récupération de données

| Technologie | Usage |
|-------------|-------|
| **TanStack Query** | Récupération côté client |
| **oRPC** | Appels RPC côté serveur |
